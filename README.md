# 📖 Overview

React Weather App is a fully responsive weather application that allows users to search for any city and instantly view real-time weather information.

The application displays weather conditions, temperature, humidity, minimum & maximum temperature, feels-like temperature, weather description, dynamic weather images, and weather icons based on current conditions.

Built using **React**, **Material UI**, and the **OpenWeather API**, this project emphasizes component-based architecture, state management, API integration, responsive UI design, and clean code organization.
